# React CRUD App

A simple CRUD (Create, Read, Update, Delete) application built with React to demonstrate basic state management, component communication, and user interaction handling. The app allows users to add, edit, delete, and sort items dynamically while persisting data using browser local storage.

---

## 🚀 Features

* Add new users/items
* Edit existing entries
* Delete items from the list
* Sort items alphabetically (A–Z / Z–A)
* Input validation
* Data persistence using **Local Storage**
* Component-based React architecture

---

## 🛠 Tech Stack

* React
* JavaScript (ES6+)
* React Hooks (useState, useEffect)
* Local Storage API
* CSS / Inline styling



