# React Mini Projects Collection

A collection of small React applications built to practice and demonstrate fundamental frontend development concepts such as state management, component architecture, form handling, API integration, and dynamic UI updates.

Each project focuses on a specific concept and is organized in separate folders with its own source code and live deployment.

---

## 🚀 Projects Included

### 1️⃣ React Counter App

A simple interactive counter application demonstrating React state management, event handling, conditional rendering, and side effects using React Hooks.

**Features**

* Increment / decrement counter
* Custom input-based updates
* Reset and clear functionality
* Hide / show counter
* Animated UI updates

**Live Demo**
https://omkar4042.github.io/React-mini-projects/counter-live

**Source Code**
counter/

---

### 2️⃣ React CRUD App

A basic CRUD application that allows users to create, read, update, and delete items. Data persistence is implemented using browser local storage.

**Features**

* Add items
* Edit items
* Delete items
* Sort list alphabetically
* Input validation
* Local storage persistence

**Live Demo**
https://omkar4042.github.io/React-mini-projects/curd-live

**Source Code**
curdapp/

---

### 3️⃣ React Todo App

A task management application where users can manage daily tasks with editing, filtering, and completion tracking functionality.

**Features**

* Add tasks
* Edit tasks
* Delete tasks
* Mark tasks as completed
* Filter tasks (All / Completed / Pending)
* Display remaining tasks
* Local storage persistence

**Live Demo**
https://omkar4042.github.io/React-mini-projects/todo-live

**Source Code**
todoapp/

---

### 4️⃣ React Components Practice

A beginner-friendly project focused on understanding React component structure, JSX rendering, controlled forms, and list rendering.

**Features**

* Functional components
* Controlled form inputs
* Basic form validation
* Rendering lists with map()
* Conditional rendering
