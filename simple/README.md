# React Simple Components Practice

A beginner-friendly React project created to practice core concepts such as functional components, JSX rendering, list rendering, controlled forms, and basic state management using React Hooks.

This project contains multiple small components used to understand how React applications are structured and how UI and logic interact in a component-based architecture.

---

## 🚀 Features

* Functional React components
* JSX rendering
* Rendering lists using `.map()`
* Controlled form inputs
* Basic form validation
* Conditional rendering
* Simple UI styling using inline CSS

---

## 🛠 Tech Stack

* React
* JavaScript (ES6+)
* React Hooks (useState)
* CSS / Inline styling

## ⚙️ Installation & Setup

1. Clone the repository

```id="d7kp33"
git clone https://github.com/Omkar4042/React-mini-projects.git
```

2. Navigate to the simple project folder

```id="gpyqqj"
cd React-mini-projects/simple
```

3. Install dependencies

```id="tpkibp"
npm install
```

4. Start the development server

```id="1d3mp2"
npm start
```
