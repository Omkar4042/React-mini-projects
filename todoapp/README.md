# React Todo App

A simple and interactive Todo application built with React that allows users to manage daily tasks efficiently. The project demonstrates core React concepts such as state management, list rendering, conditional rendering, form handling, and local storage for data persistence.

Users can add, edit, delete, complete, and filter tasks while maintaining a clean and responsive user interface.

---

## 🚀 Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed or pending
* Filter tasks (All / Completed / Pending)
* Display remaining task count
* Inline editing for tasks
* Data persistence using **Local Storage**

---

## 🛠 Tech Stack

* React
* JavaScript (ES6+)
* React Hooks (useState, useEffect)
* Local Storage API
* CSS / Inline Styling

## ⚙️ Installation & Setup

1. Clone the repository

```id="todo002"

```

2. Navigate to the Todo project folder

```id="todo003"

```

3. Install dependencies

```id="todo004"

```

4. Start the development server

