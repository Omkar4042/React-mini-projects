# React Counter App

A simple interactive counter application built with React to demonstrate core concepts such as state management, event handling, conditional rendering, and side effects using React Hooks.

---

## 🚀 Features

* Increment and decrement counter
* Multiply counter value
* Reset counter to default values
* Hide and show counter display
* Animated counter text on updates
* Dynamic background color changes based on counter value
* Controlled input fields for custom operations

---

## 🛠 Tech Stack

* React
* JavaScript (ES6+)
* React Hooks (useState, useEffect)
* CSS / Inline Styling

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/Omkar4042/React-mini-projects.git
```

2. Navigate to the counter project folder

```
cd React-mini-projects/counter
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm start
```
